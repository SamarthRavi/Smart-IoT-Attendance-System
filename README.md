# Smart IoT-Based Classroom Attendance System

## Overview

The Smart IoT-Based Classroom Attendance System is an automated attendance tracking solution developed using ESP32, Bluetooth Low Energy (BLE), Firebase Realtime Database, and a web-based dashboard.

The system automatically detects registered student devices using BLE and updates attendance records in real time. Attendance data is stored in Firebase and visualized through a live dashboard, reducing manual effort and improving classroom management efficiency.

## System Architecture

```text
Student Device
      ↓
ESP32 BLE Scanner
      ↓
Firebase Realtime Database
      ↓
Web Dashboard
```

The ESP32 continuously scans for nearby registered BLE devices. When a student device is detected, attendance information is uploaded to Firebase. The dashboard retrieves data from Firebase and displays attendance records in real time.
