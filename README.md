# Smart IoT-Based Classroom Attendance System

## Overview

This project is an IoT-based attendance system developed using ESP32, Bluetooth Low Energy (BLE), Firebase Realtime Database, and a web dashboard.

The system automatically detects registered student devices and marks attendance in real time without manual intervention.

---

## Features

- Automated Attendance Tracking
- BLE Device Detection
- Firebase Realtime Database Integration
- Real-Time Attendance Monitoring
- Cloud-Based Data Storage
- Dashboard Analytics
- Research Paper Publication
- Demonstration Video

---

## Technologies Used

- ESP32
- Bluetooth Low Energy (BLE)
- Firebase Realtime Database
- HTML
- CSS
- JavaScript
- Arduino IDE

---

## System Architecture

Student Device
↓
ESP32 BLE Scanner
↓
Firebase Realtime Database
↓
Web Dashboard
